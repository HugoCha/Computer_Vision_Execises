# Machine Learning for Defect Classification

## Statement

Manufactured parts must be classified into:

GOOD
DEFECTIVE

using extracted image features and classical machine learning methods.

This project uses 2 datasets : [good parts](data/processed/parts/good) and [defective parts](data/processed/parts/defective)

The goal is to compare rule-based vision against ML-based classification.

Models:

Logistic Regression
Support Vector Machine (SVM)
Random Forest

Evaluation:

train / test split
confusion matrix
precision / recall
false positive vs false negative analysis

## Deliverable

Example output:

Prediction: DEFECTIVE
Confidence: 92%

Visual output include confusion matrix and feature analysis plots.
